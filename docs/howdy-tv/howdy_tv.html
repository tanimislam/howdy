5. Howdy TV Functionality

These command line executables, graphical user interfaces, and APIs are used to download and manage your television shows. This documentation is organized into sections on command line tools, the GUI to visualize our TV collection ( howdy_tv_gui), the low level API, and the TODO.

The Howdy TV GUI, CLI, and API functionality is fairly robust, and I see no pressing need to add functionality here. I only see the need to fix bugs and improve the operation of current tools.
